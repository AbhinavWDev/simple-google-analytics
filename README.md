Easily integrate Google Analytics tracking code into your WordPress website without the need to edit theme files.

== Description ==

The "Simple Google Analytics" plugin provides a hassle-free way to add Google Analytics tracking code to your WordPress website. You can insert your Google Analytics tracking code (e.g., UA-XXXXXXXXX-X) through a user-friendly settings page. The plugin will then automatically inject the tracking code into your website's footer, ensuring accurate and reliable analytics data collection.

Key features of the "Simple Google Analytics" plugin:

- Easy-to-use settings page for adding your Google Analytics tracking code.
- Automatically inserts the tracking code into the footer of your website.
- No need to manually edit theme files or worry about losing your tracking code during theme updates.

The "Simple Google Analytics" plugin streamlines the process of integrating Google Analytics, allowing you to focus on analyzing website traffic and user behavior without the technical headaches.

== Installation ==

1. Upload the "simple-google-analytics" folder to the `/wp-content/plugins/` directory.
2. Activate the "Simple Google Analytics" plugin through the 'Plugins' menu in WordPress.
3. Go to the "Google Analytics" menu in the WordPress admin to enter your tracking code.

== Frequently Asked Questions ==

**Q: Where can I find my Google Analytics tracking code?**
A: You can find your Google Analytics tracking code by signing in to your Google Analytics account. Once logged in, navigate to the Admin section, select the property you want to track, and locate the "Tracking Info" section, where you'll find your tracking code (e.g., UA-XXXXXXXXX-X).

**Q: Can I use this plugin with multiple Google Analytics properties?**
A: At this time, the "Simple Google Analytics" plugin is designed to work with a single Google Analytics property. If you need to add multiple properties, you can manually insert the tracking codes into your theme files or consider using a more advanced analytics plugin.
